# Task Simple

## Tecnologias

- React
- Redux-toolkit
- React-Router-Dom
- TailwindCSS

## Instalacion

- Clone el proyecto
- Ejecute este comando para instalar las dependencias

```
npm install
```

- Ejecute este comando para correr la aplicacion

```
npm run dev
```

- Abra el navegador el el puerto `localhost:3000`
- Enjoy
